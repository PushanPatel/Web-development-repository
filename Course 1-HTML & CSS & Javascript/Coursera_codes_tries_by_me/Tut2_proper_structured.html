This is a 3 day event. Participants will be provided with .stl files( which have been provided by Cortex Technologies Private Limited) which they have to convert to CAD files for each part and then assemble them according to the assembly stl file.

This way you declare a class and use it for particular tags only.
We have worked many times on CAD modelling but never have we actually checked how efficient the model is. We just go for trial and error form, i.e., if the system or model seems unstable after 3D printing or laser cutted parts arranged then redesign it and then again see whether it is working or not. This is a very time consuming process which leads to huge wastage of materials. So, why not learn something which could make our work efficient, and isn’t this the time when we can learn new things while in quarantine. So, Tinkerer’s Lab has come up with an event to encourage everyone to get rid of laziness and hone their CAD skills. Tinker Simulation is an event which not only focuses on CAD modelling skills but also encourages students to develop dynamic simulation and stress analysis.
